# Security Policy

`mcp-tool-card-summary` is a pure-transform library and CLI: it reads JSON files from a directory and emits a structured summary. No network listener, no remote fetch, no execution of user-supplied code.

## Supported versions

Only the latest tagged release is supported.

## Reporting a vulnerability

Please use GitHub Security Advisories for private disclosure:

- [Open a security advisory](https://github.com/mizcausevic-dev/mcp-tool-card-summary/security/advisories/new)

Do not file public issues for security reports.
